# Code of Conduct

このプロジェクトでは、誰もが安心して参加できる場を保つため、Contributor Covenantの考え方に基づく行動規範を採用します。

## Our Pledge

私たちは、年齢、体型、障がい、民族、性自認、経験、国籍、外見、人種、宗教、性的指向に関わらず、すべての参加者にとって開かれた、歓迎されるコミュニティを目指します。

## Expected Behavior

- 思いやりと敬意を持って接する
- 建設的なフィードバックを行う
- 異なる意見や経験を尊重する
- プロジェクトとコミュニティにとって最善の行動を心がける

## Unacceptable Behavior

- 攻撃的、差別的、性的、威圧的な発言や行動
- ハラスメント
- 個人情報の無断公開
- 継続的な妨害行為
- その他、メンテナーが不適切と判断する行為

## Enforcement

メンテナーは、この行動規範に反する行為に対して、注意、コメントの編集または削除、IssueやPull Requestのロック、参加制限など、必要な対応を行うことがあります。

## Attribution

この行動規範は [Contributor Covenant](https://www.contributor-covenant.org/) の考え方を参考に、この小規模プロジェクト向けに簡潔化したものです。
