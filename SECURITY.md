# Security Policy

## Supported Versions

Chrome Web Store公開前のため、現時点では `main` ブランチの最新状態をサポート対象とします。

## Reporting a Vulnerability

脆弱性やプライバシー上の問題を見つけた場合は、公開Issueに詳細を書かないでください。

次のどちらかの方法で報告してください。

- GitHubのPrivate Vulnerability Reportingが有効な場合は、それを使って報告する。
- 利用できない場合は、概要だけをIssueで知らせ、詳細な再現手順や影響範囲は公開コメントに含めない。

報告時に含めてほしい情報:

- 影響を受けるバージョンまたはコミット
- 再現手順
- 期待される挙動と実際の挙動
- 影響範囲
- 可能であれば修正案

## Scope

対象範囲:

- Chrome拡張の権限利用
- note記事DOMの読み取り処理
- クリップボードへのコピー処理
- ユーザーのページ内容やプライバシーに関わる挙動

対象外:

- note.com自体の脆弱性
- ブラウザやOSの脆弱性
- ユーザーが手動で貼り付けた先のサービスでの扱い
