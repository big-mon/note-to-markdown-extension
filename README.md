# note Markdown Copier

noteの記事ページに「Markdownコピー」ボタンを追加するChrome拡張です。表示中の記事タイトル、メタ情報、見出し画像、記事本文をMarkdownとしてクリップボードへコピーします。

## 使い方

1. Chromeで `chrome://extensions/` を開きます。
2. 右上の「デベロッパー モード」を有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」を押します。
4. このリポジトリのフォルダを選択します。
5. `https://note.com/*/n/*` の記事ページを開くと、右下に「Markdownコピー」ボタンが表示されます。

## コピーされる内容

Markdownの先頭には次のメタ情報が入ります。

```markdown
# 記事タイトル

- URL: https://note.com/...
- Author: 著者名
- Published: 公開日時

![見出し画像](https://...)
```

その後に、ブラウザ上で表示されている記事本文が続きます。画像は `![alt](url)`、リンクは `[text](url)` として出力します。

## 対象範囲

- 対象ページは `https://note.com/*/n/*` の記事ページです。
- コピー対象は、現在ブラウザ上で表示できている記事本文です。
- ヘッダー、検索窓、フッター、タグ、著者プロフィール、応援UIなどの記事外要素は除外します。
- 有料・非公開などで画面に表示されていない本文は取得しません。
